Comparison on word similarity task¶

We are comparing Spearman's Rank correlation coefficient computed between : human judged similarity of word pairs which is present in datasets and corresponding word pairs embeddings cosine similarity. We are using this datasets : https://github.com/mfaruqui/eval-word-vectors/tree/master/data/word-sim

def plot_comparison(results):
    plt.figure(figsize = (10,10))
    for key in results:
        result = results[key]
        y = [result[dataset] for dataset in datasets]
        plt.plot(datasets, y, marker = 'o', label=key)
    plt.ylabel('rho')
    plt.xlabel('Dataset')
    plt.yticks(np.arange(0,1.1,0.1))
    plt.xticks(rotation=90)
    plt.legend(loc='upper right')
    plt.grid(True)
    plt.show()
