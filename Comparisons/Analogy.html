Comparison on analogy solving task¶

We used following method to solve analogy:

If given analogy problem is Word1 : Word2 -> Word3 : ? then,

? = Word having highest cosine similarity with embeddings(Word2) - embeddings(Word1) + embeddings(Word3)

We are calculating accuracy on the dataset as :

Accuracy = ( Number of analogies solved correctly / Total number of analogies )*100 .

We are evaluating on this dataset: http://download.tensorflow.org/data/questions-words.txt

def plot_comparison(results):
    plt.figure(figsize=(10,10))
    for key in results:
        result = results[key]
        y = [result[category] for category in categories]
        plt.plot(categories, y, marker = 'o', label = key)
    plt.ylabel('Accuracy %')
    plt.xlabel('Category')
    plt.yticks(np.arange(0, 50 ,5))
    plt.legend(loc='upper right')
    plt.grid(True)
    plt.show()
